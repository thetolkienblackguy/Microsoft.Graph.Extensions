# Get-GraphApplication.md

`Get-GraphApplication` is a PowerShell function that retrieves application information from the Microsoft Graph API.

## Synopsis

This function retrieves application information from the Microsoft Graph API based on application ID, display name, or a filter query.

## Syntax

```powershell
Get-GraphApplication
  [-ApplicationId] <string[]>
  [[-Select] <string[]>]
  [[-ApiVersion] <string>]

Get-GraphApplication
  -Filter <string>
  [[-Select] <string[]>]
  [[-ApiVersion] <string>]
  [[-Top] <int>]

Get-GraphApplication
  -All
  [[-Select] <string[]>]
  [[-ApiVersion] <string>]
  [[-Top] <int>]
```

## Description

The `Get-GraphApplication` function provides a flexible way to retrieve application information using the Microsoft Graph API. It supports querying by application ID, applying filters, or retrieving all applications. The function allows customization of the returned properties and the API version used.

## Parameters

- `-ApplicationId <string[]>`
  The application ID of the application(s) whose information is to be retrieved. Aliases: AppId, Id.

- `-Filter <string>`
  The filter query to be used to retrieve application information.

- `-All <switch>`
  Retrieve all applications.

- `-Select <string[]>`
  The properties to be selected for the application information. Default: DisplayName, AppId, Id.

- `-ApiVersion <string>`
  The version of the Microsoft Graph API to use. Valid values are "Beta" and "v1.0". Default: "v1.0".

- `-Top <int>`
  The number of results to return. Valid range: 1-999.

## Examples

### Example 1: Retrieve application by ID
```powershell
Get-GraphApplication -ApplicationId "12345678-1234-1234-1234-123456789012"
```

### Example 2: Retrieve applications with a filter
```powershell
Get-GraphApplication -Filter "startswith(displayName,'Test')"
```

### Example 3: Retrieve all applications with specific properties
```powershell
Get-GraphApplication -All -Select "DisplayName","AppId","Id","SignInAudience"
```
